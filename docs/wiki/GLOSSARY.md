# Глоссарий

- **Artifact** — итоговый файл или промежуточный результат рендера.
- **Asset freeze** — фиксация локальных копий и checksums всех media.
- **Beat** — смысловой/визуальный фрагмент сценария.
- **Bundle** — собранный web-проект композиции, передаваемый renderer.
- **Caption page/group** — одна одновременно отображаемая группа слов.
- **Composition** — зарегистрированный видео-шаблон с размерами, fps и duration.
- **Deterministic render** — одинаковое состояние кадра при одинаковом input/environment.
- **Frame-derived state** — визуальное состояние, вычисляемое по номеру кадра.
- **Hero frame** — кадр максимальной видимости и ясности сцены.
- **Idempotency** — повторный идентичный запрос не создает новый лишний render.
- **MCP** — Model Context Protocol для связи AI host/client с tools/resources/prompts.
- **Premounting** — ранний mount элемента до появления, чтобы assets успели загрузиться.
- **Render manifest** — параметры, версии, hashes и QA-статус конкретного результата.
- **Safe area** — зона, которую не перекрывает UI площадки и где текст безопасно читается.
- **Seekable animation** — анимация, корректно вычисляемая при переходе сразу на произвольное время.
- **Sub-composition** — вложенная самостоятельная композиция.
- **Timeline** — временная структура анимации/клипов.
- **Transition overlap** — период одновременной видимости соседних сцен.
- **Visual regression** — сравнение контрольных кадров с эталонами.
