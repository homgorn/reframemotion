# Project memory

This directory is concise durable context for Codex and maintainers.

- [FACTS.md](FACTS.md) — verified facts and pinned versions.
- [DECISIONS.md](DECISIONS.md) — architecture decision log.
- [BACKLOG.md](BACKLOG.md) — deferred tasks.
- [WIKI-INDEX.md](WIKI-INDEX.md) — index of long-form domain knowledge.

Rules:

- date claims that may change;
- distinguish verified fact from proposal;
- do not copy secrets, customer exports or generated media;
- update decisions when architecture changes, not after memory is lost.
